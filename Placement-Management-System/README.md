# 🎓 Placement Management System

A full-stack **Placement Management System** built using the **MERN Stack** to simplify and automate campus placement activities. The application allows students to register, log in, manage their profiles, and view placement opportunities, while administrators can efficiently manage students, companies, and placement records.

---

## 🚀 Features

- 🔐 Secure Login & Registration
- 👨‍🎓 Student Dashboard
- 👨‍💼 Admin Dashboard
- 🏢 Company Management
- 📋 Student Management
- 📄 Student Profile Management
- 🔍 Search & Filter Functionality
- 🧭 Client-side Routing using React Router DOM
- 📱 Responsive User Interface
- ⚡ Fast Performance using React + Vite
- 📊 Placement Status Tracking

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/Anassk-ds/Placement-Management-System.git
```

### Navigate to the Project

```bash
cd Placement-Management-System
```

### Install Dependencies

```bash
npm install
```

### Start the Development Server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```
